# common-man
Java utils


## build.gradle 
1. Set repositories
    ```gradle
    allprojects {
      repositories {
        ...
        maven { url 'https://jitpack.io' }
      }
    }
    ```
2. Set dependencies
    ```gradle
    dependencies {
      implementation 'com.github.avaj-java:common-man:0.4.8'
    }
    ```
